# Managr

A premium sober living operations app for house managers and facility owners.

## What It Does
- Manage multiple homes and all residents in one place
- Track chores, drug tests, medications, documents, and appointments
- Resident profiles that follow them — even across transfers or readmissions
- Role-based access: Admin, Manager, Viewer
- Nightly reports and incident tracking built in

## Stack
- Next.js 14 (App Router)
- Supabase (database, auth, file storage)
- Tailwind CSS + shadcn/ui
- Deployed on Vercel
