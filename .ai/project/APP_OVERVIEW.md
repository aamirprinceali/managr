# Managr — App Overview

## What it is
A web app for sober living / recovery housing operations. Built as a gift for Mike (friend who runs Lighthouse sober living). Long-term goal: sell as SaaS to other treatment facilities.

## Who uses it
- **Owner (Mike)** — sees all homes, manages team, sends messages, reviews reports
- **House Managers** — log into their assigned home, manage residents, log drug tests, complete tasks

## Tech Stack
- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4 + shadcn/ui (Base UI)
- Supabase (PostgreSQL + Auth)
- Vercel (deployment target)

## Local dev
```bash
cd ~/Desktop/dev/managr
npm run dev
# → http://localhost:3000
```
